# <a name="whats-new-in-content-moderator"></a>Content Moderator の新機能

サービス内の新機能について説明します。 以下の項目には、リリース ノート、ビデオ、ブログの投稿、およびその他の種類の情報が含まれる可能性があります。 このページをブックマークして、常にサービスの最新情報を確認してください。

## <a name="august-2020"></a>2020 年 8 月

### <a name="new-video-moderation-ui"></a>新しいビデオ モデレーション UI

* レビュー ツールのビデオ モデレーション ページが、ビデオやビデオの一部を簡単にタグ付けして処理できるようにするための多くの新しいツールと機能によって更新されています。 詳細については、[ビデオ モデレーション ガイド](./video-moderation-human-review.md)に関するページを参照してください。

### <a name="new-dashboard"></a>新しいダッシュボード

* レビュー ツールのダッシュボードが、オープン、保留中、完了済みのレビューについてより多くの情報を表示するように更新されました。 新しい機能の紹介については、[レビュー ツール ガイド](./Review-Tool-User-Guide/human-in-the-loop.md)に関するページを参照してください。
