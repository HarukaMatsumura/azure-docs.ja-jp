## <a name="reference-docs"></a>リファレンス ドキュメント

Speech サービスからは 2 つの SDK が提供されます。 最初の SDK は主要な [Speech SDK](../speech-sdk.md) であり、Speech サービスとのやりとりに必要な機能の大部分が提供されます。 2 番目の SDK はデバイスに固有であり、それにふさわしい [Speech Devices SDK](../speech-devices-sdk.md) という名前が付いています。 いずれの SDK もさまざまな言語で利用できます。

### <a name="speech-sdk-reference-docs"></a>Speech SDK リファレンス ドキュメント

適切な Speech SDK リファレンス ドキュメントを探す際、次のリストをご利用ください。

- <a href="https://aka.ms/csspeech/csharpref" target="_blank" rel="noopener">C# SDK </a>
- <a href="https://aka.ms/csspeech/cppref" target="_blank" rel="noopener">C++ SDK </a>
- <a href="https://aka.ms/csspeech/javaref" target="_blank" rel="noopener">Java SDK </a>
- <a href="https://aka.ms/csspeech/pythonref" target="_blank" rel="noopener">Python SDK</a>
- <a href="https://aka.ms/csspeech/javascriptref" target="_blank" rel="noopener">JavaScript SDK</a>
- <a href="https://aka.ms/csspeech/objectivecref" target="_blank" rel="noopener">Objective-C SDK </a>

> [!TIP]
> Speech service SDK は頻繁に保守管理され、更新されています。 変更内容、更新内容、機能の追加を追跡するには、[Speech SDK リリース ノート](../releasenotes.md)を参照してください。

### <a name="speech-devices-sdk-reference-docs"></a>Speech Devices SDK リファレンス ドキュメント

[Speech Devices SDK](../speech-devices-sdk.md) は Speech SDK の上位集合であり、特定のデバイスの機能が拡張されています。 Speech Devices SDK をダウンロードするには、最初に[開発キットを選択する](../get-speech-devices-sdk.md#choose-a-development-kit)必要があります。
