# <a name="length-azure-cosmos-db"></a>LENGTH (Azure Cosmos DB)
 指定された文字列式の文字数を返します。  
  
## <a name="syntax"></a>構文
  
```sql
LENGTH(<str_expr>)  
```  
  
## <a name="arguments"></a>引数
  
*str_expr*  
   評価される文字列式です。  
  
## <a name="return-types"></a>戻り値の型
  
  数値式を返します。  
  
## <a name="examples"></a>例
  
  次の例では、文字列の長さを返します。  
  
```sql
SELECT LENGTH("abc") AS len 
```  
  
 結果セットは次のようになります。  
  
```json
[{"len": 3}]  
```  

## <a name="remarks"></a>解説

このシステム関数では、インデックスは使用されません。
