# <a name="customize-your-defender-for-iot-solution"></a>Defender for IoT ソリューションをカスタマイズする

このガイドでは、Defender for IoT のさまざまな設定をカスタマイズする方法について説明します。

> [!div class="checklist"]
> * ソリューションの推奨事項を構成する
> * 設定を変更する

## <a name="configure-solution-recommendations"></a>ソリューションの推奨事項を構成する

Defender for IoT ソリューションの推奨事項を構成するには、以下の手順を実行します。

1. Azure portal で **[IoT Hub]** を開きます。
1. 左側のメニューの **[セキュリティ]** の下にある **[設定]** を選択して開きます。
1. **[設定]** の下にある **[Recommended Configurations]\(推奨される構成\)** を選択します。
1. 組織およびワークフローに関連するソリューションの推奨事項を無効にするか、有効にします。

## <a name="change-settings"></a>設定を変更する

Defender for IoT の設定を管理します。

Defender for IoT の設定を構成するには、次の手順を実行します。

1. Azure portal で **[IoT Hub]** を開きます。
1. 左側のメニューの **[セキュリティ]** の下にある **[設定]** を選択して開きます。
1. **[データ収集]**** で、変更するソリューション設定を選択します。
1. どの設定画面でも、設定の変更を保存するために、上部の **[保存]** を常にクリックしてください。
