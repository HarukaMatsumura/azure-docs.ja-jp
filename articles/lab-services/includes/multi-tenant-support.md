複数の Azure AD テナントの一部である管理者またはラボ所有者は、Azure Lab Services ポータル (右上隅) からテナント間を移動できます。 

![テナントを選択する](../media/multi-tenant-support/picker.png)

テナントを選択したら、左上隅にある Azure Active Directory (Azure AD) テナント内でラボ アカウントを選択します。
