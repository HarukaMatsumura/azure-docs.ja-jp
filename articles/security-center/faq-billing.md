# <a name="billing-questions"></a>課金に関する質問

## <a name="how-does-billing-work-for-azure-security-center"></a>Azure Security Center の課金のしくみを教えてください。
Security Center は、Azure Defender を使用しても使用しなくても提供されます。

- **Azure Defender を使用しない**場合、Security Center では、Azure リソースのセキュリティ状態の表示、基本的なセキュリティ ポリシー、セキュリティに関する推奨事項、およびパートナーによるセキュリティ製品とサービスとの統合を利用できます。

- **Azure Defender** によって、セキュリティ アラート、脅威インテリジェンス、行動分析、異常検出、および脅威属性レポートを含む脅威防止機能が追加されます。 Azure Defender の30 日間の無料試用版を開始できます。 アップグレードを行うには、 [セキュリティ ポリシー](https://docs.microsoft.com/azure/security-center/security-center-pricing)で価格レベルを選択してください。 詳細については、[価格のページ](https://azure.microsoft.com/pricing/details/security-center/)を参照してください。

## <a name="how-can-i-track-who-in-my-organization-enabled-azure-defender-changes-in-azure-security-center"></a>Azure Security Center で組織内のどのユーザーが Azure Defender の変更を有効にしたかを追跡する方法
Azure サブスクリプションには、価格設定を変更する許可が与えられた管理者が複数設定されていることがあります。 変更したユーザーを見つけるには、Azure アクティビティ ログを使用します。 

詳細については、[このページ](https://techcommunity.microsoft.com/t5/Security-Identity/Tracking-Changes-in-the-Pricing-Tier-for-Azure-Security-Center/td-p/390832)を参照してください。
