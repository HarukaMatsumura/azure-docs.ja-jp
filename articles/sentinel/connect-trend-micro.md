# <a name="connect-trend-micro-deep-security"></a>Trend Micro Deep Security の接続

Trend Micro Deep Security コネクタを使用すると、Deep Security のログを Azure Sentinel に簡単に接続して、ダッシュボードの表示、カスタム アラートの作成、調査の改善を行うことができます。 これにより、組織のネットワークとシステムに関するより詳細な分析情報が得られ、セキュリティ運用機能が向上します。

[!INCLUDE [reference-to-feature-availability](includes/reference-to-feature-availability.md)]


## <a name="prerequisites"></a>前提条件

- Azure Sentinel ワークスペースに対する読み取りおよび書き込みアクセス許可。

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Azure Sentinel を Trend Micro Deep Security に接続する

1. Azure portal で、 **[Azure Sentinel]**  >  **[Data connectors]\(データ コネクタ\)** に移動し、 **[Trend Micro Deep Security]** コネクタを選択します。

2. **[Open connector page]\(コネクタ ページを開く\)** を選択します。

3. 指示に従い、次の手順に進みます。
