***開始する前に**、 *[ポリシーの種類の選択]* セレクターを使用して、設定するポリシーの種類を選択します。 Azure Active Directory B2C には、ユーザーがアプリケーションを操作する方法を定義する 2 つの方法 (定義済みの[ユーザー フロー](../articles/active-directory-b2c/user-flow-overview.md)を使用する、または完全に構成可能な[カスタム ポリシー](../articles/active-directory-b2c/custom-policy-overview.md)を使用する) があります。 この記事で必要な手順は、方法ごとに異なります。
