Azure リソースのマネージドID は、Azure Active Directory の機能です。 [Azure リソースのマネージド ID をサポートする各 Azure サービス](/azure/active-directory/managed-identities-azure-resources/services-support-msi)は、それぞれ固有のタイムラインの下で提供されます。 ご利用のリソースに対するマネージド ID の[提供](/azure/active-directory/managed-identities-azure-resources/services-support-msi)状態と[既知の問題](/azure/active-directory/managed-identities-azure-resources/known-issues)をあらかじめ確認しておいてください。
