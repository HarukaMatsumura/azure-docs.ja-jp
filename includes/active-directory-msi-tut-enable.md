システム割り当てマネージド ID の有効化は、1 クリックで行うことができます。 VM の作成中に有効にするか、既存の VM のプロパティで有効にすることができます。

![新しいストレージ アカウントを作成する](./media/active-directory-msi-tut-enable/identity.png)


**新しい VM 上でシステム割り当てマネージド ID を有効にするには:** 

1. [Azure portal にサインインする](https://portal.azure.com)

2. [システム割り当て ID を有効にして仮想マシンを作成する](/azure/active-directory/managed-identities-azure-resources/qs-configure-portal-windows-vm#system-assigned-managed-identity)
