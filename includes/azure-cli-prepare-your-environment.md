## <a name="prepare-your-environment"></a>環境を準備する

1. Azure CLI コマンドを実行するには、Azure CLI の[ローカル インストール](/cli/azure/install-azure-cli)を行うか、bash 環境を使用して [Azure Cloud Shell](/azure/cloud-shell/quickstart) を開始します。

1. ローカル インストールを使用する場合は、[az login](/cli/azure/reference-index#az-login) コマンドを使用して Azure CLI でサインインします。  ターミナルに表示される手順に従って、認証プロセスを完了します。  追加のサインイン オプションについては、「[Azure CLI を使用してサインインする](/cli/azure/authenticate-azure-cli)」を参照してください。
