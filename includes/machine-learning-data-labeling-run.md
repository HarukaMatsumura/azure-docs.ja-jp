プロジェクトを初期化すると、Azure によって実行が開始されます。 メインの **[データのラベル付け]** ページでプロジェクトを選択して、プロジェクトの詳細を確認します。

プロジェクトを一時停止または再起動するには、右上の **[実行中]** 状態を切り替えます。 データにラベルを付けることができるのは、プロジェクトが実行されているときだけです。
